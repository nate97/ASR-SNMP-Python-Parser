# Customer Consumption Toolkit

The goal of this toolset is to allow an individual to inspect customer usage data and allow them to target high consumption consumers.
This application has various tools, primarily designed for a backend that allows you to maintain a pseudo database of your customerâ€™s usage data. This toolset allows you to view and inspect various pieces of information regarding data consumption. Some of the data it allows you to view are things such as; graphs, 95th percentile, by means of sorting customers from lowest to highest, and averaged maximum data usage.

## Requirements
	Python 3 or above.
	SNMP (net-snmp-utils)
	Python3-tk (python3-tkinter)
	PIP3 packages:
		matplotlib
